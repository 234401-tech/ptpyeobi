여비뚝딱 시작 안내
===================

이 폴더에 들어 있는 파일
-----------------------
1. README.md                  ← Claude Code에게 던질 작업 지시서
2. 디자인_미리보기.html        ← 만들 화면의 디자인 (브라우저로 더블클릭)
3. 사용법.txt                  ← 이 파일


시작 단계
---------

[1단계] 디자인부터 확인
- 디자인_미리보기.html을 더블클릭해서 브라우저로 열어봅니다.
- 출장유형(자가운전/동승/공용/대중교통)을 눌러보고,
  카운터·영수증 편집기·대장 추가 버튼이 어떻게 동작하는지 확인합니다.
- 이 모양 그대로 만들면 됩니다.

[2단계] 작업 폴더 준비
- 여비뚝딱이라는 빈 폴더를 하나 만듭니다.
- 거기에 README.md를 옮겨 넣습니다.
- 같은 폴더에 참고용 자료도 같이 두면 편합니다:
  · _2026_출장처리.xlsx
  · 여비지급규정.hwpx
  · 출장증빙서류 이미지들

[3단계] Claude Code 진입
  cd 여비뚝딱
  claude

[4단계] 첫 메시지
  README.md 읽고 Phase 1부터 차례로 만들어줘.
  한 Phase 끝날 때마다 멈추고 보고해줘.


Phase 구성 (총 6단계)
---------------------
Phase 1  셋업                30분
Phase 2  백엔드 도메인        1시간
Phase 3  백엔드 API           1시간 30분
Phase 4  프론트엔드           2시간
Phase 5  OCR 통합            별도
Phase 6  XLSX export         별도


주의
----
- 한 Phase 끝나면 반드시 동작 확인 후 다음으로
- 새 기능 추가 시 README.md 하단 변경 이력에 한 줄 남기기
- 백엔드는 SQLite 사용 (망분리 환경 친화적)
- 디자인 색상 톤은 디자인_미리보기.html 그대로 유지
